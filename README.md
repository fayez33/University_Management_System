# University Management Database System

## Project Overview
This repository contains the complete MSSQL database implementation for a **University Management System**. The project was designed to streamline academic operations, focusing on data integrity, automated business logic, and a scalable relational schema.

## Key Features
* **Advanced Schema Design:** Inheritance-based model for specialized entities (Students, Instructors, and Staff).
* **Database Views:** Specialized views for simplified reporting on student transcripts, faculty workloads, and department summaries.
* **Automated Business Logic:**
    * **Triggers:** Automated validation for course enrollment and capacity.
    * **Stored Procedures:** Logic for registration and financial billing.
* **Financial Tracking:** Built-in logic for managing tuition and payments.
